# PSSMS Frontend

React + Vite frontend for SmartPark PSSMS.

## Setup

1. Run `npm install`.
2. Start the app with `npm run dev`.
3. Open `http://localhost:5173`.

## Pages

- `Car` - Add car and list registered cars.
- `ParkingSlot` - Add parking slots and display slot status.
- `ParkingRecord` - Create, update, delete parking records.
- `Payment` - Record payments after calculating fees.
- `Reports` - Daily parking payment report.
